# RCB vs DC ‚Äì Ball-by-Ball Data Analysis

Ball-by-ball data analysis of the IPL 2024 RCB vs DC match using Python, Pandas, and Matplotlib to uncover scoring trends, player performance, partnerships, and phase-wise insights.

## Project Overview
This project performs a **comprehensive ball-by-ball data analysis** of the IPL 2024 match between  
**Royal Challengers Bangalore (RCB)** and **Delhi Capitals (DC)** using **Python, Pandas, and Matplotlib**.

The goal is to extract meaningful insights related to:
- Batting performance
- Bowling efficiency
- Scoring trends
- Match phases
- Partnerships and dismissals

This project is suitable for **academic submission, data analytics portfolios, and GitHub showcasing**.

---

## Dataset Description
The dataset contains **delivery-level data** for the match.

### Dataset Columns
| Column | Description |
|------|------------|
| team | Batting team |
| over | Over number (1‚Äì20) |
| batter | Striker |
| bowler | Bowler |
| non_striker | Non-striker |
| runs_batter | Runs scored by batter |
| runs_extras | Extra runs |
| runs_total | Total runs on delivery |
| player_out | Dismissed player |
| wicket_kind | Type of dismissal |
| fielders | Fielders involved |
